# Employee Management System

A full-stack Employee Management System built using Java, JSP, Servlets, JDBC, and MySQL following MVC architecture.

## Features
- Secure admin login with session management
- Add, update, delete, and view employee records
- MVC architecture with DAO pattern
- SQL injection prevention using PreparedStatement
- Exception handling for robustness

## Technologies Used
- Java
- JSP & Servlets
- JDBC
- MySQL
- HTML & CSS

## Project Architecture
- MVC (Model–View–Controller)
- DAO Pattern

## How to Run
1. Import the project into Eclipse or IntelliJ
2. Configure Apache Tomcat server
3. Set up MySQL database using provided SQL file
4. Update database credentials in the project
5. Run the project on server
